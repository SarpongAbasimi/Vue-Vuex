# leanvuex
Just a simple Vue application which illustrates the core concepts and fundamendals of
Vue routing and state management with vuex for large applications
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
